## install

```
git clone git@github.com:shuliu/my-gulp-template.git newFolderName
npm install
bower init
bower install gsap bootstrap susy compass-breakpoint components-font-awesome --save
```

**run gulpfile.coffee**

```
npm i -D coffee-script
```
